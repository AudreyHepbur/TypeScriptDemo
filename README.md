>### TypeScript 开发环境搭建

1. **安装 Node 的运行环境**

2. **全局安装 typeScript**

   ```js
   yarn global add typescript
   ```

3. **ts-node 的安装和使用**

   ```js
   npm install -g ts-node
   ```

   安装完成后，就可以在命令中直接输入如下命令，来查看结果了

   ```js
   ts-node demo.ts
   ```
